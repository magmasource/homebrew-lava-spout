# Magmasource Lava-spout

## How do I install these formulae?

`brew install magmasource/lava-spout/<formula>`

Or `brew tap magmasource/lava-spout` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "magmasource/lava-spout"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
